🪩 DISCO LIGHT 🪩
A minimalist, lightweight web application that generates random background colors using HTML, CSS, and  JavaScript.

🚀 Features
* Generates a completely randomized background color on page refresh/click.
* Clean, minimalist interface centered around color display.
* Built entirely with native web technologies—no heavy frameworks required.

🛠️ Tech Stack
* HTML5: Semantic page layout
* CSS3: Centered layout and responsive styling
* JavaScript (ES6): Random hex/RGB logic and DOM manipulation
